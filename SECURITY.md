# Security Policy

This repository contains a static 404 page artifact (HTML/CSS/JS) intended to be served without a build step.

## Reporting a vulnerability

If you believe youâ€™ve found a security issue, please report it privately:

- Email: security@propertools.be
- Subject: `[propertools/404] Security report`

Please include:
- A clear description of the issue
- Steps to reproduce (or a minimal PoC)
- Impact assessment (what an attacker could do)
- Browser + OS details

## Scope

In scope:
- `404.html`
- `assets/css/404.css`
- `assets/js/404.js`
- `assets/gifs/404/*`
- GitHub Pages deployment of this repo

Out of scope (but we still appreciate reports):
- `propertools.be` site-wide infrastructure and unrelated pages
- Third-party browser extensions or local environment issues

## Disclosure

We prefer coordinated disclosure. If you contact us first, we will:
- Acknowledge receipt
- Work to reproduce and remediate
- Credit you (if desired)

Thanks for helping keep things coherent.
